19. SUPER ADMIN PANEL
19.1 Admin Authentication & Access
•	Separate admin login portal (/admin)
•	Role-Based Access Control (RBAC):
o	Super Admin: Full access to all features
o	Moderator: Content moderation, user management
o	Support: Ticket management, user assistance
o	Finance: Payment oversight, refunds, invoices
o	Analyst: Read-only access to reports and analytics
o	Category Manager: Category and vendor approval
o	Custom Roles: Define custom permission sets
•	Permission Matrix:
o	User management (view, edit, delete, ban)
o	Payment access (view, release, refund)
o	Content moderation (approve, reject, delete)
o	System settings (view, edit)
o	Reports access (view, export)
o	Category management (create, edit, delete)
o	Dispute resolution (view, resolve)
o	Admin user management
•	Multi-factor authentication for admin accounts
•	Session timeout for security
•	Admin activity logging (all actions tracked)
